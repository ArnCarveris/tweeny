tween< T, Ts > Class Template Reference

Detailed Description

template<typename T, typename... Ts>
class tweeny::tween< T, Ts >

The tween class is the core class of tweeny. It controls the interpolation steps, easings and durations.

It should not be constructed manually but rather from tweeny::from, to facilitate template argument deduction (and also to keep your code clean).
